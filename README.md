# 📊 Student Grade Analysis

## Overview
This project analyzes student grade data using the Pandas library. It calculates descriptive statistics, finds the highest and lowest grades, computes average scores, and identifies the top-performing student.

## Features
- 📋 Display student data
- 📈 Generate summary statistics
- ⭐ Calculate average, maximum, and minimum grades
- 🏆 Identify the top-performing student
- 📊 Group and analyze data using Pandas

## Technologies Used
- Python
- Pandas

## Project Structure
```
student-grade-analysis/
│
├── main.py
├── README.md
```

## How to Run

1. Install Pandas:

```bash
pip install pandas
```

2. Run the project:

```bash
python main.py
```

## Sample Output

- Student information
- Summary statistics
- Average grade for each student
- Highest and lowest grades
- Top-performing student
